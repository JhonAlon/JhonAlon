## Sobre mí

Curso el quinto año de Ingeniería de Sistemas en la Universidad Nacional de Ingeniería (UNI). Me gusta convertir necesidades concretas en aplicaciones sencillas de usar, con una base técnica ordenada y fácil de mantener.

Desde 2024 trabajo de manera independiente en desarrollo de software. Mi experiencia incluye una aplicación Android para el control de inventario de un negocio local y la participación en sistemas web académicos con autenticación, roles, reportes, formularios y consumo de APIs.

## Proyectos destacados

### EVA API y EVA IES · Colaboración en proyecto privado

He participado en el desarrollo de una plataforma para la gestión de evaluaciones académicas. En el backend he trabajado con una API modular, autenticación, permisos, documentación y acceso a datos; en el frontend, con formularios, paneles, reportes y flujos administrativos.

`Angular` `TypeScript` `RxJS` `PrimeNG` `Tailwind CSS` `Chart.js` `ExcelJS` `Vitest`<br>
`NestJS` `Prisma` `SQL Server` `JWT` `Passport` `Swagger` `Jest`

### [Stock POS](https://github.com/JhonAlon/stock_POS) · Aplicación en uso

Aplicación Android para gestionar inventario y ventas en un negocio local. La desarrollé con arquitectura MVVM y persistencia en SQLite para que pueda funcionar completamente sin conexión.

`Kotlin` `Android` `MVVM` `SQLite` `Material Design` `CRUD` `Offline-first`

### SIRA · Proyecto académico en desarrollo

Sistema educativo pensado para docentes, estudiantes y familias. Incluye gestión de usuarios y roles, actividades, recursos educativos, seguimiento del progreso, analítica y reportes. El proyecto sigue en desarrollo y su repositorio es privado.

`Vue.js` `Vite` `Pinia` `PrimeVue` `Axios` `Fastify` `Prisma` `MySQL` `JWT` `Helmet`

### Proyectos de CS50W

Como parte de mi formación desarrollé una interfaz de búsqueda con HTML y CSS y una enciclopedia con Django que permite crear, editar, buscar y mostrar entradas escritas en Markdown. También completé **CS50: Introduction to Computer Science** de Harvard University en 2024.

`Python` `Django` `HTML` `CSS` `Bootstrap` `Markdown2`
